How to use it

  1. You can make a post request within the API test section, in order to pass a new book entry section at [URL]/api/books, to the database and it will be returned as an object with a unique code _id.
  2. Too check for "books" and or "comments" stored on the database, you can make a get request at [URL]/api/books to retrieve an aray of all books containing title, _id, & commentcount.
  3. To get a single "book's" details you can make a get request at [URL]/api/books/{_id} to retrieve a single object of a book containing title, _id, & an array of comments (empty array if no comments present).
  4. You make a post request that stores comments with corresponding book on the MongoDB database and add comment form
  5. Tou can delete /api/books/{_id}, in order to delete a book from the collection. Simply click delete bottom to remove book posted and wait for 'delete successful' (if successful) message to return.
  6. You can also delete all books in the database.Which will return message 'complete delete successful', if successful.

Please note that nothing from this website will be cached on the clients local computer as a security measure.
